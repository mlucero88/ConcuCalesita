Detailed Description

template<typename T>
class MessageQueue< T >

Clase que encapsula el manejo de una cola de mensajes (mecanismo IPC)

Utiliza template con la condicion de que dicho template sea una estructura cuyo primer miembro sea un long con nombre mtype, y que este miembro tome solo valores enteros mayores a cero
struct <name> {
long mtype;
...
};

Constructor que crea una cola de mensajes (en caso de no existir y create es true) o se conecta a la cola (en caso de ya existir y alguno o ambos failifExists o create son false), cuyo identificador de cola se obtiene a partir de el nombre de archivo filename y el caracter character.

Precondition
El archivo filename debe existir
Parameters
filenameNombre de archivo utilizado para generar el identificador de la cola
characterCaracter utilizado para generar el identificador de la cola
createCrea la cola de mensajes en caso de no existir. Se arroja una excepcion en caso de que create sea false y la cola no exista
failIfExistsSe arroja una excepcion en caso de que create sea true y la cola ya exista
Exceptions
MessageQueueExcepcionError generado al intentar abrir el archivo o crear/conectarse a la cola

template<typename T >
void MessageQueue< T >::destroy ( ) const

Metodo para eliminar la cola de mensajes.

Note
Si la cola esta siendo usada por otros procesos y uno o varios se encuentran bloqueados en espera de MessageQueue::receiveMsg, estos seran desbloqueados.
Precondition
Tener conexion a la cola de mensajes (es decir, no fue destruida por el proceso en ejecucion o cualquier otro que haga uso de la misma cola de mensajes)
Exceptions
MessageQueueExceptionError generado al intentar destruir la cola de mensajes

ssize_t MessageQueue< T >::receiveMsg ( long  type,
T &  buffer

Metodo para desacolar un mensaje de la cola de mensajes.

Note
En caso de no haber un mensaje que cumpla las condiciones solicitadas para desacolar, el metodo se bloqueara hasta recibir un mensaje o que la cola haya sido destruida
Precondition
Tener conexion a la cola de mensajes (es decir, no fue destruida por el proceso en ejecucion o cualquier otro que haga uso de la misma cola de mensajes)
Parameters
typeIdentificador de mensaje a desacolar. En caso de ser 0, toma el siguiente mensaje de la cola, sin importar el valor mtype del mensaje. En caso de ser >0, toma el siguiente mensaje de la cola con mytpe igual a type. En caso de ser <0, toma el siguiente mensaje de la cola con menor mtype, cuyo valor de mtype es menor o igual al valor absoluto de type
bufferEstructura donde se guardara el mensaje
Returns
Cantidad de bytes copiados en buffer, sin contar el primer miembro de la estructura (mtype)
Exceptions
MessageQueueExceptionError generado al intentar desacolar el mensaje
